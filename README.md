# MuJoCo Robot Learning for Physical Reasoning

## Starting Assumptions (generous, to be removed methodically)
1. Container is rigid body with known geometry
2. Single container shape (cylinder)
3. Discrete CoM positions, (bottom 3rd, halfway, top 3rd)
4. Known tray pose
5. No perception (oracle-accessible)
6. Quasi-static transport
7. 2D planning to start
8. Success = object stays in its exact pose wrt tray frame
